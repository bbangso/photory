## 📝프로젝트 사용법

Photory 시작 방법입니다.

### Frontend

1. Install NPM packages

```bash
cd vue_client
npm i
```

2. Run server

```
npm run server
```

<br>

### Backend

1. 가상환경 실행 후 진행
2. Install packages

```bash
# 가상환경 먼저 실행 후 진행
cd django_server
pip install -r requirements.txt
```

3. Migration 진행

```
python manage.py makemigrations
python manage.py migrate
```

4. Run server

```
python manage.py runserver
```

<br>

### AI server

1. 가상환경 실행 후 진행
2. Install packages

```bash
# 가상환경 먼저 실행 후 진행
cd flask_server
pip install -r requirements.txt
```

3. Run server

```
python ai_server.py
```
